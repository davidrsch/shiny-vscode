# Shiny for Python

This is an extension to help launch [Shiny for Python](https://pyshiny.netlify.app) applications. Shiny is a Python package that is designed to make it easy to build interactive web applications with the power of Pythonâ€™s data and scientific stack.

## Features

Currently, the main/only feature of this extension is the addition of a "Run Shiny App" option to the Run button when an app.py is being edited.
